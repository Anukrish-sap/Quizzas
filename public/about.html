About Quizzas

Quizzas is a modular quiz platform designed for practice and revision. Topics and questions are stored in Supabase, allowing you to scale the platform by adding new subjects without changing the UI.

Tech

Frontend: HTML/CSS/JavaScript · Backend: Supabase (Postgres + REST)

Why this project

Built as a portfolio-ready app with clean structure and scalable data model.

Scalability

Add new topics and question banks in the database—no code changes required.

Next upgrades

Authentication, admin upload, timers, leaderboards, and analytics.
