About Quizzas

Quizzas is a modular quiz platform. Topics and questions are stored in Supabase so you can expand the system to any subject later without changing your site structure.

Tech

HTML/CSS/JavaScript • Supabase (Postgres + REST + Auth) • RLS protected admin writes

Modular topics

Add topics in the database and they instantly show on the homepage.

Quiz engine

Loads questions and options dynamically and tracks score.

Admin uploader

Admins can add questions directly from the website.
